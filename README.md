# Hit Me Up for Slack
> Hit me up with slack team name availability.

This is a [Hit Me Up][hmu] plugin for checking [Slack][slack] team name availability.

## Installation
For using the on the command line:
```shell
$ npm install --global hmu-slack
```

## Usage
```shell
$ hmu slack <...names>
```
